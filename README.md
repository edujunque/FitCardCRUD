# FitCardCRUD

###Arquitetura
* MVC 5
* bootstrap
* CodeFirst

###Linguagem
* C#
* Razor
* html/css

###First run
* ctrl+B para buildar e instalar todos os pacotes através do nuget.
* Abrir package manager console e rodar o comando "update-database -v" para criar a base de dados. (O banco de dados esta configurado para rodar em uma base local)
* ctrl+F5 para rodar o projeto no navegador.
* Acessar a pagina: "localhost:{portaLocal}/Estabelecimentoes" para acessar a pagina principal
